# Cingo Log frontend

# Pré-requisitos para execução
1. Angular CLI
# Para executar projeto de log
1. Clone o projeto.
2. Abir o terminal na pasta frontend e executar a linha de comando 
"npm install" e "ng serve".
3. Acesse http://localhost:4200.
